Hexo哔哩哔哩番剧页面插件

字数总计:266|阅读时长: 1 分钟
| 阅读量:|评论数:

介绍

为Hexo添加哔哩哔哩番剧页面，参考了hexo-douban的部分代码

安装

bash

配置

将下面的配置写入站点的配置文件 _config.yml 里(不是主题的配置文件).

yaml
  • enable: 是否启用
  • vmid: 哔哩哔哩番剧页面的 vmid(uid),如何获取？
  • title: 该页面的标题
  • quote: 写在页面开头的一段话,支持html语法
  • show: 初始显示页面：0: 想看, 1: 在看, 2: 看过，默认为1
  • loading: 图片加载完成前的loading图片

使用

  1. 前往你的 Hexo 博客的根目录
  2. 输入hexo new page bangumis
  3. 你会找到source/bangumis/index.md这个文件
  4. 修改这个文件，添加type: "bangumis"：
markdown

获取uid

登录哔哩哔哩后前往https://space.bilibili.com/页面，网址最后的一串数字就是 uid

需要将追番列表设置为公开！
