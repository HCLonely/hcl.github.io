    YouTube标签

    单个视频|播放列表|搜索查询

    例：

    Code

    参数

    以下参数只有id为必需, id必须放在youtube后面, 其他参数顺序不限

    • id(必需): 视频id | 播放列表id(PL开头) | 搜索关键词(playlist选项)
    • autoplay(可简写): 加载完视频后是否自动播放
      • 0(默认值): 不自动播放, autoplay:0或不填
      • 1: 自动播放, autoplay:1或autoplay
    • cc_load_policy(可简写): 是否显示字幕
      • 0(默认值): 不显示字幕, cc_load_policy:0或不填
      • 1: 显示字幕, cc_load_policy:1或cc_load_policy
    • color: 进度条已观看部分的颜色
      • ‘red’(默认值): color:red或不填
      • ‘white’: color:white
    • controls(可简写): 是否显示播放器控件
      • 1(默认值): 显示, controls:1或不填
      • 0: 不显示, controls:0或controls
    • disablekb(可简写): 是否启用键盘控制
      • 0(默认值): 不启用, disablekb:0或不填
      • 1: 启用, disablekb:1或disablekb
    • enablejsapi(可简写): 是否允许IFrame Player API控制播放器
      • 0(默认值): 不启用, enablejsapi:0或不填
      • 1: 启用, enablejsapi:1或enablejsapi
    • start: 指定视频从距离开头多少秒的地方开始播放, 默认从开头开始播放
      • 例: start:70: 从 01:10 开始播放
    • end: 指定视频到距离开头多少秒的地方停止播放, 默认播放到视频结束
      • 例: end:70: 到 01:10 停止播放
    • start-end: 从距离开头start秒的地方开始播放，到距离开头end秒的地方停止播放
      • 例: 10-70: 到 00:10 播放到 01:10
    • fs(可简写): 是否显示全屏观看按钮
      • 1(默认值): 显示, fs:1或不填
      • 0: 不显示, fs:0或fs
    • hl: 播放器语言，参数为两个字母的 ISO 639-1 语言代码或完整表示的语言区域
      • 例: hl:zh或hl:zh-cn
    • iv_load_policy(可简写): 是否显示视频注释
      • 1(默认值): 显示, iv_load_policy:1或不填
      • 3: 不显示, iv_load_policy:3或iv_load_policy
    • listType: 显示播放列表或搜索查询结果
      • playlist: 显示播放列表，此时id为播放列表id(PL开头), listType:playlist
      • search: 显示搜索查询结果，此时id为搜索关键词, listType:search
    • loop(可简写): 是否循环播放视频或播放列表
      • 0(默认值): 不循环播放, loop:0或不填
      • 1: 循环播放, loop:1或loop
    • modestbranding(可简写): 是否在播放器控制栏中显示 YouTube 徽标, 仍然显示 YouTube 文字标签或水印
      • 0(默认值): 显示, modestbranding:0或不填
      • 1: 不显示, modestbranding:1或modestbranding
    • width: 指定播放器宽度, 默认720
      • 例: width:560: 播放器宽度为560px
    • height: 指定播放器高度, 默认405
      • 例: height:560: 播放器高度为560px
    • width x height: 播放器大小为 width x height
      • 例: 560x560: 播放器大小为 560px x 560px
    • nofullscreen: 不允许全屏, nofullscreen
